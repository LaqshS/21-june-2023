Programming Languages

  • Java
    - A popular object-oriented programming language developed by Sun Microsystems in 1995.

  • Python
    - A high-level, interpreted programming language created by Guido van Rossum in 1991.

  • C++
    - A general-purpose programming language created by Bjarne Stroustrup in 1983 as an extension of the C programming language.

  • JavaScript
    - A scripting language used primarily for web development, created by Brendan Eich in 1995.

  • PHP
    - A server-side scripting language used for web development, created by Rasmus Lerdorf in 1994.

  • Ruby
    - A dynamic, object-oriented programming language created by Yukihiro Matsumoto in 1995.

  • Swift
    - A general-purpose, compiled programming language developed by Apple in 2014 for iOS, macOS, watchOS, and tvOS development.

  • Go
    - A statically typed, compiled programming language designed at Google in 2007.

  • Perl
    - A high-level, general-purpose programming language created by Larry Wall in 1987.

  • Rust
    - A systems programming language developed by Mozilla in 2010.
